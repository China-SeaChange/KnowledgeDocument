# InkDoc 开源文档模板

简洁、交互丰富的纯 HTML 文档模板，适用于项目文档、技术手册、个人知识库等。无需构建，即开即用。

## ✨ 特性

- 📄 **纯静态 HTML** – 单个文件，除两个 JS 库外无额外依赖，可离线使用。
- 🎨 **简约黑白风格** – 干净清爽，专注内容阅读。
- 🔍 **实时搜索与高亮** – 基于 mark.js，实时匹配并高亮关键词，支持点击跳转。
- 📂 **可折叠导航** – 左侧章节可折叠/展开，桌面端支持拖拽调整宽度。
- 📱 **移动端友好** – 侧滑菜单、响应式布局，完美适配手机/平板。
- 📊 **ECharts 集成** – 内置图表示例，轻松展示数据，支持交互（tooltip、图例切换）。
- 🔍 **全局缩放** – 右下角 1x / 2x 按钮，适配不同屏幕或阅读距离。
- ⚡ **交互细节** – 平滑滚动、锚点跳转、回到顶部、闪烁高亮。

## 🚀 快速开始

1. 下载 [`index.html`](./index.html) 到本地。
2. 用现代浏览器（Chrome、Firefox、Edge、Safari）打开。
3. 开始浏览或编辑内容。

或者克隆仓库并启动本地服务器：

```bash
git clone https://github.com/yourname/seachange-inkdoc.git
cd seachange-inkdoc
# 使用 Python 启动临时服务器
python -m http.server 8000
```

访问 `http://localhost:8000`。

## 📁 文件结构

单一 HTML 文件，所有样式、结构、脚本均内联。核心内容位于 `<body>` 中，通过 CSS 变量 `--base-size` 控制全局缩放。

## 📘 使用指南

### 导航与阅读
- **折叠/展开**：点击左侧章节标题可收起/展开子项。
- **跳转**：点击导航链接平滑滚动到对应标题。
- **宽度调整（桌面）**：拖拽左侧栏右边缘改变宽度（240px～500px）。
- **移动端菜单**：点击顶部 `☰` 打开侧边栏，点击遮罩关闭。

### 搜索
- 输入关键词，实时下拉显示匹配段落（范围包括标题、段落、列表项、表格单元格）。
- 点击结果，页面滚动至对应元素，关键词被高亮并闪烁黄色背景。
- 搜索忽略大小写，支持特殊字符转义。

### 图表交互
- 示例饼图支持 tooltip、图例切换（ECharts 原生功能）。
- 可在 `<script>` 中修改 `initChart` 的 `option` 自定义数据和样式。

### 缩放与返回顶部
- **浮动按钮**（桌面端显示）：
  - `1x`：恢复默认大小（缩放系数 1.0）
  - `2x`：放大至 1.5 倍（便于阅读）
  - `▲`：快速回到右侧内容区顶部
- 移动端自动隐藏缩放按钮（使用系统手势替代）。

## 🛠 自定义修改

本页面本身就是模板实例，所有内容均可替换。**小白可直接将 HTML 交给 AI，结合自己的资料生成新文档，需要修改哪里直接告诉 AI。**

### 替换内容
- 正文位于 `<div id="mainContent">` 内，替换文字、表格、图片即可。
- 注意每个标题的 `id` 需与左侧导航 `<a href="#...">` 保持一致。

### 修改图表
- 每个图表容器有唯一 `id`，在 `<script>` 末尾找到 `initChart` 调用，修改 `option`。
- 示例饼图数据位置：
  ```js
  initChart('demoPie', {
      series: [{
          data: [
              { name: '结构', value: 45 },
              { name: '使用', value: 32 },
              { name: '自定义', value: 23 }
          ]
      }]
  });
  ```

### 调整样式
- 全局缩放系数：修改 `:root` 的 `--base-size` 默认值（`1`）。
- 颜色、间距等：直接覆盖 CSS 变量或重写样式。
- 移动端断点：修改 `@media (max-width: 768px)` 的像素值。

### 修改导航
- 左侧导航结构位于 `.left-nav` 内，每个 `.nav-section` 对应一个折叠块。
- 添加/删除 `<li><a href="#id">文本</a></li>` 即可，注意 `href` 与目标 `id` 匹配。

### 扩展搜索范围
- 编辑 `getSearchableElements` 函数中的 `querySelectorAll` 参数，例如增加 `'th'`、`'pre'`。

## 🌐 部署到静态托管

### GitHub Pages
1. 上传 `index.html` 到仓库。
2. 进入 Settings → Pages，选择 main 分支保存。
3. 访问 `https://<用户名>.github.io/<仓库名>`。

### Vercel / Netlify
直接拖拽上传文件夹，或连接 Git 仓库自动部署。

### 云存储（OSS/COS）
开启静态网站功能，上传文件并绑定域名即可。

## 📦 依赖

- [mark.js](https://markjs.io/) v8.11.1 – 搜索高亮
- [ECharts](https://echarts.apache.org/) v5 – 图表库

均通过 CDN 引入。如需离线使用，可下载到本地并修改 `<script>` 中的 `src` 路径。

## 🌍 浏览器支持

- ✅ 现代桌面浏览器（Chrome、Firefox、Edge、Safari 最新版）
- ✅ 移动端 iOS Safari / Android Chrome
- ❌ **不支持 IE11**（使用了 CSS 变量和 ES6）

## ⚠️ 已知限制

- 拖拽改变左侧栏宽度后，ECharts 不会自动 resize（刷新页面即可）。
- 搜索未做防抖，但性能足够应对常规文档。
